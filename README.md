Este projeto foi desenvolvido como parte do meu processo de aprendizado em desenvolvimento web, 
com foco na construção de uma página de conteúdo utilizando HTML e CSS. O objetivo principal foi 
praticar a estruturação semântica do HTML, organização de layout e estilização com CSS, explorando 
conceitos como responsividade, hierarquia visual e boas práticas de código. A página foi criada de 
forma simples, porém funcional, servindo como base para consolidar conhecimentos essenciais e evoluir 
para projetos mais complexos no futuro.
